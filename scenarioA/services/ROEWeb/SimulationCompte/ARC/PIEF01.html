Consentement pour partager les renseignements avec Service Canada

La fonction énoncé personnel d'identité est un service qui vous aide à vous inscrire à un programme gouvernemental qui requert des renseignements personnels pour authentifier votre identité. Cette fonction vous permet de partager certains des renseignements personnels que vous avez fournis à l'Agence du revenu du Canada (ARC) avec Service Canada (SC). Service Canada utilisera ces renseignements pour authentifier votre identité.

Veuillez examiner les renseignements énumérés dans cet énoncé. Si vous les jugez exacts, signez-les électroniquement et donnez à l'ARC votre consentement de les transférer à Service Canada. Les renseignements seront fournis uniquement pour authentifier votre identité. Service Canada pourra les conserver pour une période de temps en conformité avec les lois applicables et aux politiques de confidentialité.

En cliquant sur « J'accepte » ci-dessous, vous signez électroniquement vos rensignements et vous donnez à l'ARC votre consentement de les divulguer à Service Canada, seulement pour authentifier votre identité. Si vous clickez sur « Je refuse », vous retournerez au site web de Service Canada et metterez ainsi fin à la session.

Si l'un ou l'autre des renseignements ci-dessus est inexact, appelez 1-800-959-8281.

Téléimprimeur - Demandes de renseignements pour les personnes ayant une déficience auditive ou un trouble de la parole : 1-800-665-0354.

Additional security feature preference

If you select "Ask me a security question each time I login using this device," each time you login you will be asked to provide the answer to one of the questions that you chose.

If this is a trusted device and you choose "Do not ask me a security question each time I login using this device," a persistent cookie will be added to your browser. As a result, when you login you will not have to answer any of the questions that you chose.

Note

We recommend that you select "Do not ask me a security question each time I login using this device" only for your personal device and not one that is shared by others - for example, in Internet cafés or public libraries.
